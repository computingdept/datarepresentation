OCR J277 – Image Size & Colour Depth (Auto Check)

Answer what the question asks for — bits, bytes and/or KB. Use simple conversions: ÷ 8 and ÷ 1000.

Rules to use
  • Image file size (bits) = width × height × colour depth
  • Sound file size (bits) = sample rate × sample resolution × time (seconds) × channels
  • Bytes = bits ÷ 8
  • Kilobytes (KB) = bytes ÷ 1000
  • Colour depth = log2(number of colours) (e.g., 16 colours → 4 bits)
  • Channels: mono = 1, stereo = 2
